# setTheme()

Sets chart theme tokens using CSS custom properties

## Usage

``` r
setTheme(io, text_color = NULL, grid_color = NULL, bg = NULL, font = NULL, ...)
```

## Arguments

- io:

  an htmlwidget object created by the myIO() function

- text_color:

  text and label color

- grid_color:

  grid line color

- bg:

  background color

- font:

  font family

- ...:

  additional CSS custom property overrides without the \`chart-\` prefix

## Value

the same myIO object
